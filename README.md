# EnviroVitals
Capstone Project to collect water and air quality data in real-time displayed on a webserver

Air quality sensors include:
- Ozone (O<sub>3</sub>)
- Nitrogen Dioxide (NO<sub>2</sub>)
- Fine Particulate Matter (PM 2.5)
